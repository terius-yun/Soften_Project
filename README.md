Soften_Project
--------------------
java, jsp 로 구현한 더미텍스트

프로젝트 설명
--------------------
쉽고 간편하게 사용자의 니즈에 맞게 선택한 주제의 더미 텍스트를 제공함으로써

업무 효율 및 작업물의 완성도를 높일 수 있는 프로그램

사용 설명
--------------------
사용자가 주제와 필요한 텍스트의 양을 선택한 후 텍스트 만들기를 진행한다

생성된 텍스트를 즉시 복사할 수 있으며 버튼을 통해 그 양을 추가하거나 줄일 수 있다
